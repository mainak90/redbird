#Very Simple webserver instance to serve as a reverse proxy for node applications in PROD/DR
##Made using RedBird Modulem please find here : https://github.com/OptimalBits/redbird 

To install --> 
1) Git clone 
2) npm install
3) pm2 start app.js

##If pm2 is not installed, first run "npm install pm2 -g" and then try command 3.
